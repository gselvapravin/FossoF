FossoF
======
Cubietruck-Debian
=================

Scripts to create an Image of FossoF-Debian Wheezy for Cubieboard 2 and Cubietruck




Installation steps
------------------

```shell
sudo apt-get -y install git
cd ~
git clone https://github.com/gselvapravin/FossoF
chmod +x ./FossoF/fossof.sh
cd ./FossoF
sudo ./fossof.sh
```
